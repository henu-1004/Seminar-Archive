Torchtune : 파이토치 기반의 하이퍼파라미터 튜닝도구
pip install torch torchvision torchtune

Pytorch 프레임워크를 사용 딥러닝 모델의 하이퍼파라미터를 자동으로 튜닝하는 라이브러리
pytorch는? 오픈소스 딥러닝 라이브러리, 자연어처리(nlp) 및 다양한 인공지능, 머신러닝 작업을 수행하는데 사용. >> 자동미분, 대규모데이터처리(데이터로더, 데이터셋클래스), gpu가속(cuda옵션으로 가능), 모듈형설계 등의 장점, numpy의 ndarray(n-dimensional array)를 tensor로 대체가능 >> ai쪽에 특화

파인튜닝:미세조정, 사전 학습된 인공지능 모델의 가중치를 새로운 데이터에 맞게 세밀하게 조정하여 성능을 향상시키고 학습 시간을 줄이는 과정이다.
허깅페이스 : 최신모델 가중치모음을 제공하는 허브
Torchtune? >> LLM을 쉽게작성, 파인튜닝하고 실험할수있는 네이티브 pytorch라이브러리.
Llama2와 Mistral모델을 지원, full finetuning, LoRA, QLORA등 다양한 파인튜닝 메소드를 지원 
 HuggingFace Datasets와의 통합, 분산 훈련 지원, 다양한 파인튜닝 기법에 대한 훈련 레시피 및 벤치마크 제공 등을 특징으로 합니다.

python3 -m venv myenv
source myenv/bin/activate
deactivate
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install torchtune

파인튜닝하는거
 tune run lora_finetune_single_device --config llama3/8B_lora_single_device


LoRa란 > Low_Rank Adaptation of Large Language Models : 대형언어모델을 효율적으로 파인튜닝 하기 위한 기법 중 하나. 파라미터 수와 계산비용을줄이면서 모델의 성능을 유지하거나 향상 가능. 
-저차원 행렬로의 근사 : 큰모델의 파라미터변화를 저차원 행렬곱으로 근사한다. 즉 파라미터 행렬의 일부를 저차원으로 분해하여 모델을 업데이트함.
-모델의 일부만 업데이트:모든파라미터를 업데이트 하는 대신, 특정 층이나 모듈의 일부 파라미터만 업데이트함 > 학습속도를 높이고 메모리 사용량을 줄임
-기존모델의 성능 유지 : 원래모델의 성능을유지하면서 효율적으로 파인튜닝할수있도록 설계

양자화 : 신호처리분야에서 사용되는 용어. 연속적인 신호나 값을 불연속적인 값/레벨로 변환하는것을 의미했음/ 딥러닝도 유사한 개념임. 연속적인 부동소수점 가중치 값을 불연속적인 정수값으로 전화하는것. >> 모델의 크기를줄이고 연산속도를 높이고 전력소모를 줄일수있음 >> 자원이 적은 환경(서버가 아닌엣지디바이스>핸드폰 등)에서 모델을 돌릴때 필수임
