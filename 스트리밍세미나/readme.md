# Streaming Seminar

뉴스 사용자 행동 데이터를 기반으로 스트리밍 데이터 처리와 그래프 기반 피처 구성 방식을 학습한 세미나 자료입니다.

본 자료에서는 Kafka의 기본 구조를 학습하고, 뉴스 클릭 로그와 뉴스 메타데이터를 활용하여 유저-뉴스 그래프를 구성하는 실험을 진행했습니다.

## Topics

- Apache Kafka 기본 구조 학습
- Producer / Broker / Consumer 개념 정리
- 뉴스 행동 로그 기반 데이터 처리
- User-News bipartite graph 구성
- NetworkX 기반 그래프 생성
- BERT embedding 기반 뉴스 텍스트 피처 생성
- PyTorch Geometric 기반 GCN 실험
